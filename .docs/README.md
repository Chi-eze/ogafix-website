# OgaFix Hidden Documentation

This directory contains sensitive documentation and procedures that shouldn't be in the main repository.

## Contents

- **DISASTER_RECOVERY.md** - Complete disaster recovery procedures
- **EC2_SETUP.md** - EC2 instance setup guide
- **CREDENTIALS.md** - Credentials and secrets management
- **MONITORING.md** - Monitoring and alerting setup
- **BACKUP.md** - Backup and restore procedures
- **TROUBLESHOOTING.md** - Common issues and solutions

## Access

These files are hidden from the main repository view but available for reference during emergencies.

To view:
```bash
cat .docs/DISASTER_RECOVERY.md
cat .docs/EC2_SETUP.md
```

## Important
